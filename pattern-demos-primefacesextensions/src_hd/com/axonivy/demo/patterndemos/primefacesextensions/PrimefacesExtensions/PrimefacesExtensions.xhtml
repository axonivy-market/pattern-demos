Primefaces Extensions Demo
The demo shows a change to InputText to restrict the number of bytes (instead of the nubmer of characters) allowed. The widget below allows entering text containing special characters (Umlaut, Emojis) which could require more than one byte per character. Nevertheless, it will not allow you to enter text, that exceeds 10 bytes of storage space. For the demo, the widget changed to red-color in case the number of characters was restricted by the extension function.
