# Pattern Demos

Pattern Demos sind eine Sammlung von Prozessen, Dialogen, Code und anderen Ausschnitten, die in Projekten nützlich sein können.

### Hauptmerkmale

- Sichtbares Fehlerhandling mit einem wiederverwendbaren AdminTask-Pattern für manuelle Wiederherstellung.
- Wiederverwendbare UI-Controller, die Zustände über Dialoge hinweg synchronisieren.
- Leichter Locking-Service zur Vermeidung paralleler Ausführungen und Race-Conditions.
- Flexibles Job-Pattern mit Scheduler, manueller Ausführung und admin-gesteuerten Retry-Workflows.
- Muster für parallele Aufgaben und zum Unterbrechen/Weiterführen von Prozessen per externem Ereignis.
- Hilfsfunktionen und Beispiele: Platzhalter-Ersetzung, PDF-Anzeige, ZIP-Verarbeitung und PrimeFaces-Erweiterungen.
